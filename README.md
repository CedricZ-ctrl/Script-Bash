# 🛠️ Linux Bash Scripts & Automation Lab

Welcome to my personal repository of Bash scripts! 🚀

This repository serves as my **dynamic cheatsheet** and learning lab for Linux system administration. 

## 🎯 Project Goal
The idea is to centralize all the tools I develop to simplify server management and automate redundant tasks. This is a living project that evolves as I gain more technical experience.

## 💡 Philosophy & Learning Journey
> [!IMPORTANT]
> **Note on my expertise:** I do not consider myself a Bash expert. This code reflects my learning path: you might find errors or methods that could be optimized. 
> 
> My goal is to bridge the gap between theory (courses, exercises) and real-world practice. I am committed to continuous improvement and learning by doing!

## 📂 Repository Structure
* **/Administration**: Deployment and config scripts (Samba, Wireguard, User management).
* **/Monitoring**: Health check tools (Disk space, open ports) and Hypervisor management.
* **/Training**: Exercises covering RegEx, `sed`, and Bash scripting fundamentals.

## 🚀 How to use
1. Clone the repo: `git clone https://github.com/CedricZ-ctrl/Script-Bash.git`
2. Grant execution rights: `chmod +x folder/script.sh`
3. Execute with caution (always review the code before running it!).
